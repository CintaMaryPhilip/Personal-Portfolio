Projects

COMPARATIVE EVALUATION OF VARIOUS SINGLE-PHASE PWM TECHNIQUES ON FPGA/DSP BASED 5-LEVEL CHB INVERTER

This project focuses on the design and implementation of a single-phase 5-level Cascaded H-Bridge (CHB) inverter for photovoltaic (PV) applications, using FPGA-based control. It evaluates and compares multiple PWM techniques—Sinusoidal PWM (SPWM), Selective Harmonic Elimination PWM (MSHE), and Bus Clamping PWM (BCPWM)—in terms of Total Harmonic Distortion (THD), switching losses, and overall efficiency. Developed in Verilog for FPGA compatibility, the system ensures low-latency control and real-time adaptability. The inverter design is optimized for industrial standards, offering enhanced harmonic reduction, reduced EMI, and improved performance for PV integration.

OPTIMIZING PARTIAL DISCHARGE DETECTION THROUGH ULTRASOUND AND DEEP LEARNING

Optimizing Partial Discharge Detection through Ultrasound and Deep Learning This project leverages ultrasound sensing and deep learning to enhance the detection and classification of Partial Discharge (PD) faults in electrical equipment. Three custom-built Convolutional Neural Network (CNN) models were developed to accurately identify four PD types—arcing, corona discharge, tracking, and looseness—along with healthy conditions. Trained on real-world noisy data, the models achieved over 94% accuracy, demonstrating strong resilience and reliability in practical environments. This work highlights the potential of deep learning in predictive maintenance and fault diagnosis for high-voltage equipment.

HAMMING CODE

Implemented a (7, 4) Hamming Code algorithm on a breadboard using logic gate ICs to detect and correct single-bit errors in a 4-bit data stream. This hardware-based project demonstrates real-time error detection and correction, enhancing data reliability in digital communication systems.

AT89C51 Parallel Programmer

Designed and developed a parallel programmer for the AT89C51 microcontroller, enabling efficient and reliable flashing of firmware onto the device. This project streamlines the programming process, ensuring accurate and consistent microcontroller updates for embedded applications.
